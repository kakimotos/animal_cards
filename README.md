## このアプリは何？

動物を写真と一緒に紹介しているモバイルアプリです。

yahoo!キッズ動物図鑑の情報をお借りして、flutter 学習のため個人利用しています。

## どこでアプリが見れる？

Web サーバーには上げていません。

以下の手順でローカルにて確認できます。

- Android Studio や VSCode を flutter 開発のためにセットアップ
- このリポジトリをクローン
- エミュレータ、シミュレータを立ち上げ確認

環境構築の参考資料：https://docs.flutter.dev/get-started/install
